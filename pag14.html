funcion: es una plataforma de soporte tecnico via web que incluye chat en vivo y gestion de clientes.

requerimientos del sistema:

  • Sistema Operativo: Kayako es compatible con múltiples sistemas operativos, incluyendo Windows, macOS, Linux, android e ios

  • Navegador Web: este generalmente es accesible a través de un navegador web moderno como Google Chrome, Mozilla Firefox, Safari o Microsoft Edge

  • Conexión a Internet: Se requiere una conexión a Internet estable y de banda ancha para acceder a la plataforma y gestionar las interacciones con los clientes

  • Dispositivo Compatible: Puedes acceder a Kayako desde una computadora de escritorio, laptop, tableta o dispositivo móvil.
