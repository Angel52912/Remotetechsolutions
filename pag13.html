funcion: es una pagina web que permite la interaccion en tiempo real con los visitantes mediante un chat en vivo

requerimientos del sistema:

  • Navegador Web: este generalmente es accesible a través de un navegador web moderno como Google Chrome, Mozilla Firefox, Safari o Microsoft Edge

  • Conexión a Internet: Se requiere una conexión a Internet estable y de banda ancha para acceder a la plataforma y mantener las comunicaciones en tiempo real

  • Dispositivo Compatible: Puedes acceder a LivePerson desde una computadora de escritorio, laptop, tableta o dispositivo móvil

  • Sistema Operativo: LivePerson es compatible con múltiples sistemas operativos, incluyendo Windows, macOS, Linux, iOS y Android
