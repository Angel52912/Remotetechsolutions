Mantenimiento de oportunidad:

que es el que aprovecha las paradas o periodos de no uso de los equipos para realizar las operaciones de mantenimiento, realizando las revisiones o reparaciones necesarias para garantizar el buen funcionamiento de los equipos en el nuevo periodo de utilización

ejemplos:

Realizar actualizaciones del sistema operativo y software durante la noche o los fines de semana cuando la PC no se está utilizando, minimizando así la interrupción de las actividades diarias.

Aprovechar un momento en que la PC está siendo abierta para añadir nueva memoria RAM o un disco duro adicional para realizar una limpieza interna del polvo y verificar conexiones de cables.

Configurar el sistema para que realice copias de seguridad automáticas de los datos durante periodos en los que el usuario no esté activo, como la hora del almuerzo o reuniones.

Ejecutar herramientas de verificación y reparación de disco (como CHKDSK en Windows) durante el arranque o reinicio programado del sistema cuando no se necesite el uso inmediato de la PC.

Mientras se instala nuevo software, también se pueden desinstalar programas no utilizados, limpiar archivos temporales y realizar una desfragmentación del disco duro en segundo plano.

Actualizar los controladores de dispositivos como la tarjeta gráfica o la tarjeta de red durante la hora del almuerzo o después de finalizar la jornada laboral.

Realizar pruebas de seguridad y análisis de vulnerabilidad de la red mientras se llevan a cabo otras actividades de mantenimiento planificadas.

Al instalar un nuevo componente de hardware, como una tarjeta gráfica o un SSD, aprovechar para verificar el estado de otros componentes y realizar limpieza de contactos y ventiladores.
