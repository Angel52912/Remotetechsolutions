Mantenimiento preventivo:

como el destinado a garantizar la fiabilidad de equipos en funcionamiento antes de que pueda producirse un accidente o avería por deterioro, este se divide en activo y pasivo

ejemplos:

Abrir la carcasa de la PC y eliminar el polvo acumulado con aire comprimido para evitar el sobrecalentamiento y mantener una ventilación adecuada.
Limpiar el teclado, el ratón y la pantalla con productos adecuados para evitar la acumulación de suciedad y bacterias.

Mantener el sistema operativo actualizado con los últimos parches y actualizaciones de seguridad para proteger contra vulnerabilidades

Asegurarse de que todas las aplicaciones y programas instalados estén actualizados a sus versiones más recientes para mejorar la seguridad y el rendimiento.

Verificar y actualizar regularmente los controladores de dispositivos como la tarjeta gráfica, la tarjeta de red, y otros componentes para asegurar un funcionamiento óptimo y solucionar posibles problemas de compatibilidad


Realizar desfragmentación periódica en discos duros mecánicos para mejorar la eficiencia del almacenamiento y el acceso a los datos


Utilizar herramientas como CHKDSK en Windows para verificar y reparar errores en el disco duro, asegurando la integridad de los datos almacenados.
Copia de Seguridad:

Realización de copias de seguridad regulares: Configurar y ejecutar copias de seguridad automáticas de datos importantes para protegerse contra la pérdida de datos por fallos de hardware, errores humanos o ataques de malware.
Optimización del Sistema:

Utilizar herramientas como el Liberador de espacio en disco o CCleaner para eliminar archivos temporales, cachés y otros archivos innecesarios que ocupan espacio y pueden ralentizar el sistema.
Gestionar y desactivar programas innecesarios que se inician al arrancar el sistema para mejorar el tiempo de arranque y el rendimiento general.

Implementar herramientas de monitoreo del sistema para vigilar la temperatura del CPU, el uso de la memoria, y otros parámetros críticos para detectar problemas potenciales antes de que causen fallos.

Ejecutar análisis completos del sistema con software antivirus y antimalware para detectar y eliminar amenazas de seguridad.
Revisión de políticas de seguridad: Asegurarse de que las políticas de seguridad, como las contraseñas y las configuraciones de firewall, estén actualizadas y adecuadamente configuradas.
