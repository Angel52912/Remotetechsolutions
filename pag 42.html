seguridad informatica

¿Que es?

La seguridad informática se refiere a proteger los sistemas de computación y datos contra accesos no autorizados, daños o cualquier otro tipo de amenaza

ejemplos:

Hardware:

Firewalls: Dispositivos que controlan el tráfico de red y aplican reglas de seguridad


Dispositivos de autenticación: Como tokens de seguridad o llaves USB para autenticación de dos factores


Sistemas de detección y prevención de intrusiones (IDS/IPS): Equipos que monitorean y responden a actividades sospechosas en lasredes


Cámaras de seguridad: Utilizadas para la vigilancia de áreas sensibles como centros de datos


Dispositivos criptográficos: Como módulos de seguridad hardware (HSM) para manejar claves y operaciones criptográficas de manera segura


Software:

Antivirus y antimalware: Programas diseñados para detectar y eliminar software sospechosos


Firewalls de software: Aplicaciones que controlan el tráfico de red en un SO


Sistemas de gestión de vulnerabilidades: Herramientas que escanean sistemas en busca de vulnerabilidades y sugieren parches


Software de encriptación: Para cifrar datos sensibles y proteger su confidencialidad en los datos


Herramientas de monitoreo de seguridad: Software que monitorea logs y actividades del sistema en busca de comportamientos sospechosos
