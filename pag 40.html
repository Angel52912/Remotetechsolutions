Dispositivos de Redes


Routers:

Dispositivos que encaminan datos entre diferentes redes, determinando la mejor ruta para el tráfico.

tipos de routers:

Router de Banda Ancha:

Conecta a Internet a través de conexiones de banda ancha como DSL, cable o fibra óptica.

Uso en: Hogares y pequeñas oficinasPrecio: $50 - $200


Router Inalámbrico (Wi-Fi):

Proporciona conectividad inalámbrica a dispositivos en una red local.

Uso en: Hogares, oficinas pequeñas y medianas.

Precio:
Banda Simple: $30 - $100
Banda Dual: $50 - $200
Banda Tri: $100 - $300


Router de Malla (Mesh):

Usa varios nodos para cubrir áreas grandes con una red Wi-Fi consistente.

Uso en: Hogares grandes, oficinas y espacios públicos.

Precio: $200 - $500 (por sistema)


Router para Gaming:

Diseñado para minimizar la latencia y optimizar la conexión para juegos en línea.

Uso en: Hogares con jugadores exigentes.

Precio: $100 - $400


Router Empresarial:

Ofrece características avanzadas de seguridad, gestión de tráfico y capacidad para manejar múltiples conexiones simultáneas.

Uso en: Pequeñas, medianas y grandes empresas.

Precio: $200 - $2000


Router de Viaje:

Compacto y portátil, ideal para asegurar conexiones en hoteles u otras ubicaciones temporales.

Uso en: Viajeros frecuentes y trabajadores remotos.

Precio: $30 - $100


Router LTE/5G:

Utiliza redes móviles para proporcionar conectividad a Internet.

uso en: Áreas rurales, ubicaciones temporales, y como respaldo para conexiones principales.

Precio: $100 - $400


Router VPN:

Integra capacidades de VPN para crear conexiones seguras entre redes.

Uso en: Pequeñas y medianas empresas, usuarios con alta preocupación por la seguridad.

Precio: $100 - $500


Router Modular:

Permite la adición de módulos para funciones específicas, como seguridad avanzada, Wi-Fi adicional o soporte de más conexiones.

Uso en: Grandes empresas y data centers.

Precio: $500 - $3000 (más módulos adicionales)


Switches:

Conectan dispositivos dentro de la misma red y envían datos solo al dispositivo destinatario correcto.

tipos de switches:

Switch No Gestionable (Unmanaged Switch):

Dispositivos plug-and-play que no requieren configuración. Proporcionan conectividad básica entre dispositivos en una red local (LAN)


Uso en: Redes pequeñas, oficinas domésticas o para expandir temporalmente una red


Switch Gestionable (Managed Switch):

Ofrecen características avanzadas de administración, seguridad y control de tráfico. Se pueden configurar, gestionar y monitorear a través de una interfaz de usuario


Uso en: Redes empresariales, data centers, y entornos donde se requiere un control detallado sobre la red


tipos de Switches Gestionables:


Switch Gestionable de Nivel Básico (Smart Switch):

Ofrecen algunas funciones de administración avanzadas, como la creación de VLANs y QoS (Quality of Service), pero no son tan complejos como los switches gestionables de nivel empresarial


Uso en; Pequeñas y medianas empresas que necesitan más control que un switch no gestionable pero no requieren todas las funciones avanzadas de un switch gestionable completo


Switch Gestionable de Nivel Empresarial (Fully Managed Switch):

Ofrecen un conjunto completo de características de administración, incluyendo enrutamiento de capa 3, VLANs, QoS avanzado, mirroring de puertos, y soporte para SNMP


Uso en: Grandes empresas, data centers, y entornos críticos donde se necesita máxima flexibilidad y control


Tipos de Switches Basados en la Arquitectura de la Red:


Switch de Capa 2:

Opera en la capa de enlace de datos (capa 2) del modelo OSI, utilizando direcciones MAC para enviar datos a través de la red


Uso en: Redes LAN para conectar dispositivos dentro de la misma red


Switch de Capa 3:

Opera en la capa de red (capa 3) del modelo OSI, capaz de realizar tareas de enrutamiento utilizando direcciones IP además de las funciones de conmutación de capa 2


Uso: Redes más grandes y complejas que necesitan segmentación de subredes y capacidades de enrutamiento internas


Tipos de Switches Basados en la Configuración Física:


Switch Modular:

Permiten agregar o quitar módulos para aumentar el número de puertos o agregar funcionalidad adicional como enrutamiento o puertos de alta velocidad


Uso en: Entornos empresariales y data centers donde se requiere flexibilidad y escalabilidad


Switch Fijo:

Tienen una configuración fija de puertos y características, sin capacidad de expansión mediante módulos


Uso en: Pequeñas y medianas empresas donde la capacidad actual satisface las necesidades y no se anticipa un crecimiento significativo


Tipos de Switches Basados en la Tecnología de Puertos:


Switch Ethernet:

La forma más común de switch, que utiliza la tecnología Ethernet para transmitir datos


Uso en: Redes LAN de todo tipo, desde hogares hasta grandes empresas


Switch Gigabit Ethernet:

Soporta velocidades de 1 Gbps por puerto


Uso en: Entornos donde se requiere alta velocidad de transmisión, como oficinas y data centers


Switch Fast Ethernet:

Soporta velocidades de 100 Mbps por puerto


Uso en: Redes más antiguas o de menor demanda de ancho de banda


Switch 10 Gigabit Ethernet:

Soporta velocidades de 10 Gbps por puerto


Uso en: Data centers y redes empresariales de alto rendimiento


Switches Especializados:


Switch PoE (Power over Ethernet):

Proveen energía eléctrica a través de cables Ethernet, lo que permite alimentar dispositivos como cámaras IP, teléfonos VoIP y puntos de acceso inalámbricos sin necesidad de cables de alimentación adicionales


Uso en: Redes donde se despliegan dispositivos PoE para simplificar la instalación y reducir costos de cableado


Switch de Conmutación de Alta Densidad:

Diseñados para soportar un gran número de puertos en un solo chasis o unidad


Uso en: Data centers y grandes infraestructuras de red que requieren alta densidad de puertos


Switch de Core:

Actúan como el switch principal en el núcleo de una red, manejando grandes volúmenes de tráfico y proporcionando alta velocidad y redundancia


Uso en: Grandes redes empresariales y data centers


Switch de Edge:

Colocados en el borde de la red, conectando dispositivos finales a la red principal


Uso en: Conectar dispositivos finales como computadoras, impresoras y cámaras de seguridad en redes empresariales


Hubs:

Dispositivos que conectan múltiples Ethernet en una red, pero envían datos a todos los puertos.

tipos de hub:

Hub Pasivo

Un hub pasivo simplemente conecta los dispositivos en una red sin amplificar o regenerar las señales que se transmiten entre ellos


Uso en: Generalmente utilizado en redes pequeñas donde la distancia entre los dispositivos conectados es corta y la pérdida de señal no es significativa


Hub Activo:

Un hub activo amplifica o regenera las señales que pasan a través de él, ayudando a extender la distancia sobre la cual los datos pueden viajar sin degradación de la señal


Uso en: redes donde los dispositivos están más alejados unos de otros o donde hay una mayor necesidad de mantener la integridad de la señal


Hub Inteligente:

Un hub inteligente (a veces llamado hub gestionable) incluye algunas capacidades de administración, como monitoreo básico de tráfico y diagnósticos simples. Aunque no es tan avanzado como un switch gestionable, ofrece más funciones que un hub pasivo o activo


Uso en: redes pequeñas o medianas donde se requiere un cierto grado de monitoreo y control, pero no se necesita la complejidad de un switch gestionable


Tipos de Hubs Basados en la Tecnología de Conexión:


Hub Ethernet:

El tipo más común de hub, utilizado para conectar dispositivos en una red Ethernet


Uso en: Redes locales (LAN) en hogares, pequeñas oficinas y algunas configuraciones empresariales simples


Hub USB:

Permite la conexión de múltiples dispositivos USB a un solo puerto USB en una computadora


Uso en: Ampliar la cantidad de puertos USB disponibles para dispositivos como teclados, ratones, impresoras y otros periféricos


Hub Especializado:


Diseñado para ser montado en un rack de servidor, facilitando la organización y gestión de múltiples conexiones en un entorno de data center o sala de servidores


Uso en: Entornos empresariales y data centers donde la organización y gestión de múltiples dispositivos de red es crucial.


Access Points (AP):

Proveen conectividad inalámbrica a dispositivos dentro de una red.

tipos de access points:


Punto de Acceso Autónomo:

Funcionan de manera independiente y se configuran individualmente. No dependen de un controlador central


Uso en: Redes pequeñas o medianas donde no se requiere gestión centralizada


Punto de Acceso Controlado:

Dependientes de un controlador central que gestiona y configura múltiples puntos de acceso simultáneamente


Uso en: Redes grandes, como en campus o grandes empresas, donde se necesita gestión centralizada y eficiente


tipos de Puntos de Acceso Basados en Tecnología Inalámbrica:


Punto de Acceso de Banda Simple (Single Band):

Opera en una sola banda de frecuencia, típicamente la banda de 2.4 GHz
Uso en: Ambientes donde la demanda de ancho de banda es moderada y la interferencia de otros dispositivos no es un gran problema


Punto de Acceso de Banda Dual (Dual Band):

Opera tanto en la banda de 2.4 GHz como en la banda de 5 GHz, ofreciendo mayor flexibilidad y menor interferencia.
Usado en: Ambientes con alta densidad de dispositivos y necesidad de mayor ancho de banda, como oficinas y hogares con múltiples dispositivos conectados


Punto de Acceso de Banda Tri (Tri Band):

Opera en una banda de 2.4 GHz y dos bandas de 5 GHz, permitiendo aún más dispositivos conectados con menor interferencia.
Usado en: Entornos de alta densidad y alta demanda de ancho de banda, como grandes oficinas y centros de convenciones.


Tipos de Puntos de Acceso Basados en la Configuración Física:


Punto de Acceso para Interiores (Indoor Access Point):

Diseñados para uso en interiores, con características adecuadas para oficinas, hogares y establecimientos comerciales


Usado para: Conectar dispositivos dentro de edificios, como computadoras, teléfonos inteligentes, y otros dispositivos IoT


Punto de Acceso para Exteriores (Outdoor Access Point):

Diseñados para resistir condiciones climáticas adversas, con protección contra agua y polvo


Usado para: Proveer conectividad en exteriores, como campus universitarios, parques industriales, y espacios públicos


Punto de Acceso con Power over Ethernet (PoE):

Reciben energía eléctrica a través del mismo cable Ethernet que transmite los datos, eliminando la necesidad de cables de alimentación separados


Usado para: Simplificar la instalación en ubicaciones donde es difícil o costoso llevar cables de alimentación


Modems:

Convertidores entre señales digitales y analógicas, permitiendo la conexión a internet a través de líneas telefónicas o cable.

tipos de modems:

Módem de Dial-Up:

Utiliza una línea telefónica estándar para conectarse a Internet. Es el tipo más antiguo de módem.
Velocidad: Máximo de 56 Kbps.

Uso en: Áreas rurales o como respaldo para conexiones principales.


tipos de Módem de Dial-Up:

Módem Interno: Instalado dentro de una computadora, generalmente en una ranura PCI o PCIe.

Módem Externo: Conectado a la computadora a través de un puerto serie o USB.


Módem DSL (Digital Subscriber Line):

Utiliza líneas telefónicas para transmitir datos a altas velocidades sin interferir con las llamadas telefónicas.

Velocidad: De 256 Kbps a varios Mbps, dependiendo del tipo (ADSL, SDSL, VDSL).

Uso en: Hogares y pequeñas oficinas.


tipos de Módem DSL:


ADSL (Asymmetric Digital Subscriber Line): Velocidades de descarga mayores que las de subida.


SDSL (Symmetric Digital Subscriber Line): Velocidades de subida y descarga iguales.


VDSL (Very High Bitrate Digital Subscriber Line): Velocidades muy altas a distancias cortas.


G.fast: Alta velocidad en distancias cortas, cercano a la fibra óptica.


Módem de Cable:


Conectado a través de la red de televisión por cable, ofrece velocidades más rápidas que DSL.
Velocidad: Hasta varios cientos de Mbps.


Uso en: Hogares y pequeñas oficinas.


tipos de Módem de Cable:


DOCSIS 2.0: Velocidades de hasta 38 Mbps de descarga.


DOCSIS 3.0: Velocidades de descarga de hasta 1 Gbps.


DOCSIS 3.1: Velocidades de descarga superiores a 10 Gbps.


Módem de Fibra Óptica:


Utiliza cables de fibra óptica para ofrecer velocidades extremadamente altas y baja latencia.


Velocidad: Hasta 1 Gbps o más.


Uso en: Hogares, oficinas y negocios que requieren alta velocidad.


tipos de Módem de Fibra Óptica:


GPON (Gigabit Passive Optical Network): Hasta 2.5 Gbps, usado en FTTH (Fiber to the Home)


EPON (Ethernet Passive Optical Network): Usa Ethernet para transmitir datos.


XG-PON: Evolución de GPON, ofrece 10 Gbps.


Módem Satelital:


Utiliza satélites para proporcionar acceso a Internet, ideal para áreas remotas.


Velocidad: 1 Mbps a 25 Mbps.
Uso: Áreas rurales, embarcaciones, ubicaciones remotas.


tipos de Módem Satelital:


Banda Ku: Frecuencias de 12 a 18 GHz.


Banda Ka: Frecuencias de 26.5 a 40 GHz.


Banda C: Frecuencias de 4 a 8 GHz.


Módem 3G/4G/5G:


Utiliza redes de telefonía móvil para proporcionar acceso a Internet


Velocidad: 3G hasta 2 Mbps, 4G hasta 100 Mbps, 5G hasta varios Gbps


Uso en: Conexiones móviles, hotspots portátiles


tipos de Módem 3G/4G/5G:


Módem USB: Conecta a un puerto USB para acceso móvil.


Módem MiFi: Punto de acceso móvil portátil.

 


Módem Integrado: En dispositivos como laptops y tablets.


Router LTE/5G: Router con módem móvil integrado.


Módem ISDN (Integrated Services Digital Network):


Proporciona transmisión digital a través de líneas telefónicas


Velocidad: Hasta 128 Kbps


Uso: Conexiones a Internet y servicios telefónicos en negocios


tipos de Módem ISDN:


Módem Básico (BRI - Basic Rate Interface): Dos canales B de 64 Kbps y un canal D de 16 Kbps


Módem Primario (PRI - Primary Rate Interface): 23 canales B de 64 Kbps y un canal D de 64 Kbps en América del Norte y Japón, o 30 canales B y un canal D en Europa


Módem de Banda Ancha Inalámbrica (WiMAX):


Proporciona acceso a Internet de banda ancha a través de señales inalámbricas de largo alcance


Velocidad: Hasta 70 Mbps


Usado en: Áreas urbanas y rurales con infraestructura de cableado limitada


tipos de Módem de Banda Ancha Inalámbrica:


Fijo: Para acceso a Internet fijo en ubicaciones específicas


Móvil: Proporciona acceso en movimiento, permitiendo conexiones desde varios lugares

Topologías de Redes:


Topología de Bus:

Todos los dispositivos están conectados a un único cable central.
Topología de Estrella:

Todos los dispositivos están conectados a un dispositivo central (por ejemplo, un switch).
Topología de Anillo:

Cada dispositivo está conectado a dos otros dispositivos, formando un anillo cerrado.
Topología de Malla:

Cada dispositivo está conectado a muchos otros dispositivos, proporcionando múltiples rutas para los datos.
Topología de Árbol:

Jerarquía de dispositivos en la que múltiples dispositivos están conectados a un nodo central



Protocolos de Cableados (T568A y T568B):


T568A y T568B son estándares de cableado para la terminación de cables de par trenzado en conectores RJ-45, utilizados en redes Ethernet.

T568A:

Pin 1: Blanco/Verde
Pin 2: Verde
Pin 3: Blanco/Naranja
Pin 4: Azul
Pin 5: Blanco/Azul
Pin 6: Naranja
Pin 7: Blanco/Marrón
Pin 8: Marrón


T568B:

Pin 1: Blanco/Naranja
Pin 2: Naranja
Pin 3: Blanco/Verde
Pin 4: Azul
Pin 5: Blanco/Azul
Pin 6: Verde
Pin 7: Blanco/Marrón
Pin 8: Marrón


Clases de Direcciones IP:


Las direcciones IP se dividen en varias clases, cada una diseñada para diferentes tipos de redes:

Clase A: Para redes muy grandes.

Rango: 1.0.0.0 a 126.0.0.0
Número de redes posibles: 128 (pero sólo se usan 126)
Número de hosts por red: 16,777,214
Clase B: Para redes medianas.

Rango: 128.0.0.0 a 191.255.0.0
Número de redes posibles: 16,384
Número de hosts por red: 65,534
Clase C: Para redes pequeñas.

Rango: 192.0.0.0 a 223.255.255.0
Número de redes posibles: 2,097,152
Número de hosts por red: 254

 

Clase D: Para multicast.

Rango: 224.0.0.0 a 239.255.255.255

Clase E: Reservadas para usos futuros.

Rango: 240.0.0.0 a 255.255.255.255
